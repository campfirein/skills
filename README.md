# byterover skill

Public, auditable install surface for the ByteRover memory skill.
Generated from the private `byterover-mono` (the `apps/skill` app) and
published on each release — **do not edit here**. Start at
[`SKILL.md`](./SKILL.md); the deterministic, zero-LLM runtime is in
[`scripts/`](./scripts).

```bash
npx skills add campfirein/skills@skill-v0.2.0
```

Released as `skill-v0.2.0` from `byterover-mono@4d57df8`.
